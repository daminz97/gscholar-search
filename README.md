# gscholar-search

### Environment Setup:
```
python 3.9
scholarly 1.7.11
```

### Usage
Search by author name|author ID|publication title
```
python main.py [--name="Author Name"]|
               [--id="Author gscholar ID"]|
               [--title="Publication Title"] [--bib="Generate bibtex"] [--output="Output Path"]
```
